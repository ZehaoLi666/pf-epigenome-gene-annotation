# Archive

This directory contains historical scripts and placeholder files that were retained for provenance but are not part of the supported public workflow.

Contents include:

- `legacy_r/`: figure-generation and exploratory R scripts with lab-specific assumptions.
- `legacy_scripts/`: helper Python and shell scripts that depended on local storage layouts or one-off analysis environments.
- `placeholders/`: empty or incomplete draft files that are not runnable.

These files are preserved as reference only. The published method is documented from the repository root `README.md`.
